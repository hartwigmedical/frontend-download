Hartwig Medical Foundation Download Portal

Files can be downloaded by creating https URLs, valid during a limited period for security reasons. These URLs can be used to download (via browser or command line tools) or, in the case of indexed files like the BAM format, directly read (streamed) by software like IGV (Integrative Genome Browser).

Notes

  • Files are potentially large (up to 300GB) so please be aware when selecting multiple!
  • Session time is 4 hours, after this period you will have to login again.
  • The URLs are valid for 7 days, after this period you will have to re-create URLs.
  • This portal contains molecular data only, no clinical information (for data requests that require clinical information please see our Data Policy)

Please let us know if you have comments or want to share your portal experience. For more information and contact please visit our website.

Usage examples

There are three options to obtain files. When you select one or more datasets/rows (by checking the checkboxes), a button "Checkout Runs" pops up at the top right above the table. Then you can select file-types and press next to get the following options:

  • Download Aria2 Config (for downloading with Aria2, recommended)
  • Download Files (for downloading from browser directly)
  • Copy links to clipboard (for using URLs in any tool that handles https).

Example download using aria2:

We recommend to use aria2 for downloading. The files are then automatically saved to a separate directory per dataset and an md5 checksum is automatically performed for file integrity. Aria2 is also capable of resuming interrupted downloads, any many more options are described at their website.

  • Download the config file (use the "Download Aria2 Config" option)
  • Start a download with for example the following code:
aria2c -x 5 -c -i aria2.txt

The "-x 5" option means 5 downloads will be started simultaneously, "-c" allows aria2 to resume a stopped download and "-i" is to provide the input file with URLs. The resulting log provides a summary of download and file integrity check results.

Example download using browser:

Use the option "Download files" to directly start a download in your browser, this is not recommended for large files.

Example loading BAM file in IGV:

  • Create URLs for BAM and BAI file (option "Copy Links")
  • Open IGV and choose option "Load from URL"
  • Paste the BAM file link at field "File URL"
  • Paste the BAI file link at field "Index URL"
