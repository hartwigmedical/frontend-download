Hartwig Medical Foundation Download Portal

This portal allows viewing and download of generated data. For each sample/patient certain files can be downloaded by creating https links. These links are only valid during a limited period for security reasons and can be used to download (via browser or command line tools) or, in the case of indexed files like the BAM format, directly read (streamed) by software like IGV (Integrative Genome Browser).

Details

  • Files can be very large (up to 300GB) so please be aware when selecting multiple sets!
  • Session time is 4 hours, after this period you will have to login again
  • Links are valid for 7 days, after this period you will have to re-create the links
  • This portal contains "molecular" data, no clinical information (for data requests that require clinical information please see our Data Policy)
  • Examples of available data formats are BAM, gVCF and VCF
  • See below for some usage examples

Please let us know if you have comments or want to share your portal experience. For more information and contact please visit our website.

Usage examples

There are three options to retrieve files. When you use the checkboxes to select one or more datasets/rows these options appear at the top right above the table:

  • Download Aria2 Config
  • Download Files
  • Copy Links

Example download using aria2:

For downloading files using command line we recommend to use the aria2 download tool. Files are saved to a separate directory per analysis and a md5 checksum is automatically performed for file integrity. Aria2 is also capable of resuming interrupted downloads, you can visit their website to check all options.

  • download the config file (use the "Download Aria2 Config" option)
  • and start a download with the following code:
aria2c -V -x 5 -c -i aria2.txt

The "-x 5" option means 5 downloads will be started simultaneously, "-c" allows aria2 to resume a stopped download and "-i" is to provide the input file with links.

Example download using browser:

Use the option "DOWNLOAD FILES" to directly start a download in your browser, for example to quickly view a pdf or other small file.

Example loading BAM file in IGV:

  • create links for BAM and accompanying BAI file (use the "COPY LINKS" option)
  • open the IGV program and choose option "Load from URL"
  • paste the BAM file link at field "File URL"
  • paste the BAI file link at field "Index URL"
