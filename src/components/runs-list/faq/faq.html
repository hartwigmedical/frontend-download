Hartwig Medical Foundation Download Portal

The portal

This portal allows view/download of data from our analysis pipeline. For each sample/patient certain files can be downloaded by creating links (valid for 24 hours). The links can be used to download (via browser or command line tools) or, in the case of indexed files like the BAM format, directly read (stream) by software like IGV (Integrative Genome Browser).

Good to know

  • Files can be very large (up to 300GB) so please check before starting a download!
  • The session time is 4 hours, after this period you will have to login again
  • Links are valid for 24 hours, after this period you will have to create new links
  • This portal contains "molecular" data, no clinical information (for data requests that require clinical information please see our Data Policy)
  • Examples of available data formats are BAM, gVCF and VCF
  • See below for some usage examples

Please let us know if you have comments or want to share your portal experience. For more information and contact please visit our website.

Usage examples

There are three options to retrieve files. When you use the checkboxes to select one or more datasets/rows these options appear at the top right above the table:

  • DOWNLOAD ARIA2 CONFIG
  • DOWNLOAD FILES
  • COPY LINKS

Example download using aria2:

For downloading files on the command line we highly recommend to use the aria2 download tool. When using this method files are saved to a separate directory per analysis and md5 checksum check is included.

  • download the config file (use the "DOWNLOAD ARIA2 CONFIG" option)
  • and start a download with the following code:
aria2c -x 5 -c -i aria2.txt

Example download using browser:

Use the option "DOWNLOAD FILES" to directly start a download in your browser, for example to quickly view a pdf or other small file.

Example loading BAM file in IGV:

  • create links for BAM and accompanying BAI file (use the "COPY LINKS" option)
  • open the IGV program and choose option "Load from URL"
  • paste the BAM file link at field "File URL"
  • paste the BAI file link at field "Index URL"
